Input form control creation.

Create a new instance of the InputForm class. The class expects 2 values - the name of the control into which it should create its content (string), and a whether or not it should support editing (boolean)

var inputform = new imqs.InputForm("content",true) ;

Add all controls that are required to the fields property on the control.The field property is an array that expects items of type InputFormField.

inputform.fields.push(new imqs.InputFormField(true,true,"Project Number:",imqs.InputFormFieldType.TEXT,1,"Project",["PR7465"],{"placeholder":"Unique Project Number"}));

Call the show method to display all controls.

inputform.show();
